About

Alcheme is Lisp inspired by Scheme (a small, extendable core) with a focus on image based programming, The intent is to helping me learn about language implementation.

Eventually I want to formalize it into a text / course how to implement a Lisp-like language on-top of WASM and maybe even a reference implementation showcasing Venn.
